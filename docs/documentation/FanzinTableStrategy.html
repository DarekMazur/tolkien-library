FanzinTableStrategy()

Concrete implementation of ITableStrategy for rendering and handling table data for Fanzin entities. Manages column aliases, header definitions, and value extraction (including nested properties) for IFanzinProps items.

# getHeaders(item)

Builds and returns the header definitions for a given item. The headers are defined in a fixed order: 'title', 'version', 'publisher', 'numbers', 'startDate', 'lastIssueDate'. Headers with a `condition` function will be included only if the condition returns true for the provided item.
